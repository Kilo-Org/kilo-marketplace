# Prerequisite Topics for Learning a Modern Graphics Engine

Students should have basic understanding of these topics before starting:

## Must Know Before Starting

| Topic | What to Google | Expected Understanding |
|-------|---------------|------------------------|
| CPU vs GPU | "CPU GPU difference" | CPU orchestrates, GPU draws |
| Vertex & Index Buffers | "vertex buffer index buffer" | Raw triangle data storage |
| Graphics Pipeline | "graphics pipeline DirectX 12" | Input assembly → rasterization → output |
| Shader Basics | "HLSL tutorial" | Vertex shader, pixel shader roles |
| Render Targets | "render target texture DirectX" | Where drawing output goes |

## Helpful But Not Required

| Topic | What to Google | Expected Understanding |
|-------|---------------|------------------------|
| Descriptor Heap | "descriptor heap DirectX 12" | How textures are bound to shaders |
| Memory Barriers | "resource barrier DirectX 12" | GPU state synchronization |
| Ray Tracing AS | "acceleration structure DirectX 12" | Speed up ray tracing |
| Meshlets | "meshlet DirectX 12" | Batching triangles for GPU |
| Compute Shaders | "compute shader tutorial" | GPU general-purpose computing |
| Samplers | "texture sampler filtering" | Bilinear, trilinear, anisotropic |

## Nice to Know

| Topic | What to Google | Expected Understanding |
|-------|---------------|------------------------|
| PBR Lighting | "PBR rendering physically based" | Modern lighting model |
| Temporal Effects | "temporal anti aliasing" | History-based effects |
| Ambient Occlusion | "SSAO screen space ambient occlusion" | Fake GI approximation |
| Path Tracing | "path tracing tutorial" | Global illumination |
