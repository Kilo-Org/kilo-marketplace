# Glossary Terms for Graphics/Rendering Engines

| Term | Plain English | Also Known As |
|------|---------------|---------------|
| **Gfx** | Graphics | - |
| **Pass** | One step in rendering pipeline | Render pass |
| **Pipeline State (PSO)** | Complete GPU configuration | Pipeline state object |
| **Descriptor** | GPU handle to bind resources | Shader resource view |
| **SRV** | Shader Resource View | Read-only resource binding |
| **UAV** | Unordered Access View | Random read/write binding |
| **CBV** | Constant Buffer View | Read-only constant binding |
| **Meshlet** | Small batch of triangles | - |
| **BLAS** | Bottom-Level Acceleration Structure | Ray tracing geometry |
| **TLAS** | Top-Level Acceleration Structure | Ray tracing instances |
| **G-Buffer** | Render targets with geometry data | Geometry buffer |
| **Hi-Z / HZB** | Hierarchical Z-Buffer | Occlusion culling |
| **DAG** | Directed Acyclic Graph | Pass dependency graph |
| **TAA** | Temporal Anti-Aliasing | - |
| **DOF** | Depth of Field | - |
| **GTAO** | Horizon-Based Ambient Occlusion | - |
| **ReSTIR** | Reservoir Spatio-Temporal Importance Sampling | GI technique |
| **NRD** | NVIDIA Ray Tracing Denoiser | - |
| **FSR2** | AMD FidelityFX Super Resolution 2 | Upscaling |
| **DLSS** | NVIDIA Deep Learning Super Sampling | Upscaling |
| **Mesh Shading** | Next-gen draw model | - |
| **Indirect Draw** | GPU-driven draw calls | - |
| **Render Graph** | DAG-based frame graph | - |
